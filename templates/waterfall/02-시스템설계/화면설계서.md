## 1. 설계 원칙

### 1.1 사용성 (Usability)

| 원칙 | 설명 | 적용 방법 |
|------|------|-----------|
| 직관성 | 사용자가 학습 없이 즉시 사용 가능 | 표준 UI 패턴 사용, 명확한 레이블 |
| 일관성 | 전체 화면에서 동일한 패턴과 규칙 적용 | 디자인 시스템 기반 컴포넌트 재사용 |
| 피드백 | 사용자 행동에 대한 즉각적 반응 | 로딩 표시, 성공/에러 메시지, 상태 변화 |
| 효율성 | 최소 단계로 목표 달성 가능 | 핵심 기능 접근 [N]클릭 이내 |
| 오류 방지 | 실수 가능성 사전 차단 | 유효성 검증, 확인 대화상자, 되돌리기 |

### 1.2 접근성 (Accessibility)

| 기준 | 수준 | 적용 항목 |
|------|------|-----------|
| WCAG | [2.1 AA] | 전체 화면 적용 |
| 키보드 접근성 | 필수 | 모든 인터랙티브 요소 Tab 이동 가능 |
| 스크린리더 | 필수 | ARIA 속성, 시멘틱 HTML |
| 색상 대비 | 최소 4.5:1 (일반 텍스트) | 전체 텍스트 및 아이콘 |
| 폰트 크기 | 최소 14px (본문) | 확대 200%까지 레이아웃 유지 |
| 대체 텍스트 | 필수 | 모든 이미지에 alt 텍스트 |
| 포커스 표시 | 필수 | 포커스 상태 시각적 표시 |

### 1.3 반응형 디자인 (Responsive)

| 브레이크포인트 | 범위 | 대상 기기 | 레이아웃 |
|--------------|------|-----------|----------|
| Mobile (sm) | < 640px | 스마트폰 | 단일 컬럼 |
| Tablet (md) | 640px ~ 1023px | 태블릿 | 2 컬럼 |
| Desktop (lg) | 1024px ~ 1279px | 노트북 | 사이드바 + 메인 |
| Wide (xl) | >= 1280px | 데스크톱 | 풀 레이아웃 |

## 3. 화면 흐름도

### 3.1 인증 플로우

```mermaid
flowchart TD
    START([앱 진입]) --> CHECK{인증 상태<br/>확인}
    CHECK -->|미인증| LOGIN[SCR-AUTH-001<br/>로그인]
    CHECK -->|인증됨| DASHBOARD[SCR-MAIN-001<br/>대시보드]

    LOGIN -->|로그인 성공| DASHBOARD
    LOGIN -->|회원가입 클릭| REGISTER[SCR-AUTH-002<br/>회원가입]
    LOGIN -->|비밀번호 찾기| FORGOT[SCR-AUTH-003<br/>비밀번호 재설정]

    REGISTER -->|가입 완료| EMAIL_VERIFY[이메일 인증 안내]
    EMAIL_VERIFY -->|인증 완료| LOGIN

    FORGOT -->|재설정 링크 발송| RESET[비밀번호 재설정<br/>입력 화면]
    RESET -->|변경 완료| LOGIN

    style LOGIN fill:#E8A838,color:#fff
    style DASHBOARD fill:#4A90D9,color:#fff
    style REGISTER fill:#5BA85B,color:#fff
```

### 3.2 게시글 플로우

```mermaid
flowchart TD
    LIST[SCR-POST-001<br/>게시글 목록] -->|게시글 클릭| DETAIL[SCR-POST-002<br/>게시글 상세]
    LIST -->|글쓰기 버튼| NEW[SCR-POST-003<br/>게시글 작성]

    DETAIL -->|수정 버튼<br/>작성자만| EDIT[SCR-POST-004<br/>게시글 편집]

    NEW -->|임시 저장| NEW
    NEW -->|발행| DETAIL
    NEW -->|취소| LIST

    EDIT -->|저장| DETAIL
    EDIT -->|취소| DETAIL

    DELETE_CONFIRM -->|확인| LIST
    DELETE_CONFIRM -->|취소| DETAIL

## 5. 예시 화면 명세

### 5.1 SCR-AUTH-001: 로그인

#### 기본 정보

| 항목 | 내용 |
|------|------|
| **화면 ID** | SCR-AUTH-001 |
| **화면명** | 로그인 |

|        +----------------------+          |

|        +----------------------+          |
|        [  비밀번호 찾기 링크  ]          |
|        +----------------------+          |
|        |     로그인 버튼       |          |
|        +----------------------+          |
|                                          |

|        |  Google 로그인 버튼   |          |
|        +----------------------+          |
|        +----------------------+          |
|        |  Kakao 로그인 버튼    |          |
|        +----------------------+          |
|                                          |

```

#### 구성 요소

| 요소 ID | 유형 | 설명 | 동작 | 검증 규칙 |

| 비밀번호 표시 토글 | 눈 아이콘 클릭 | 비밀번호 마스킹/해제 전환 | N/A |
