# vive-md 예시 프롬프트 모음

이 문서의 프롬프트를 Claude Code에 입력하면, 각 템플릿 가이드를 기반으로 코딩하거나 산출물을 생성합니다.

## 1. 기술 스택 가이드 활용 프롬프트

### Spring Boot 프로젝트

```
Spring-Boot-개발가이드.md를 CLAUDE.md로 복사하고, 이 프로젝트에 적용해줘.

이 프로젝트는 쇼핑몰 백엔드야. 다음 기능을 만들어줘:
- 회원가입/로그인 API (JWT 기반)
- 상품 CRUD API
- 주문 생성 API
가이드의 보안, 에러 핸들링, JPA 최적화 패턴을 모두 적용해.
```

```
Spring-Boot-개발가이드.md를 참고해서 글로벌 예외 처리를 구현해줘.
BusinessException 계층, @RestControllerAdvice, 에러 코드 enum까지 전부.
```

```
Spring-Boot-개발가이드.md 기반으로 User 엔티티의 CRUD를 만들어줘.
Repository, Service, Controller, DTO, 테스트까지 전부 포함.
N+1 방지, Bean Validation, 보안 패턴 모두 적용.
```

### Vue.js 프로젝트

```
Vue3-개발가이드.md를 CLAUDE.md로 복사하고 적용해줘.

이 프로젝트는 관리자 대시보드야. 다음을 만들어줘:
- 로그인 페이지 (JWT 인증)
- 사용자 관리 페이지 (목록/상세/수정)
- Pinia store로 상태 관리
가이드의 Composable 패턴, 보안, 성능 최적화 모두 적용.
```

```
Vue3-개발가이드.md를 참고해서 usePagination composable을 만들어줘.
API 연동, 로딩 상태, 에러 처리까지 포함.
```

### React 프로젝트

```
React-개발가이드.md를 CLAUDE.md로 복사하고 적용해줘.

이 프로젝트는 블로그 프론트엔드야. 다음을 만들어줘:
- 게시글 목록/상세 페이지
- TanStack Query로 서버 상태 관리
- Zustand로 클라이언트 상태 관리
가이드의 커스텀 훅 패턴, 코드 스플리팅, 접근성 모두 적용.
```

```
React-개발가이드.md를 참고해서 ProtectedRoute 컴포넌트와
인증 관련 커스텀 훅(useAuth, useUser)을 만들어줘.
JWT 토큰 갱신 로직 포함.
```

### Next.js 프로젝트

```
NextJS-개발가이드.md를 CLAUDE.md로 복사하고 적용해줘.

이 프로젝트는 이커머스 웹앱이야. 다음을 만들어줘:
- 상품 목록 페이지 (Server Component + ISR)
- 상품 상세 페이지 (generateStaticParams)
- 장바구니 (Client Component + Zustand)
- 주문 생성 (Server Action + Zod 검증)
가이드의 Server/Client 분리, 캐싱 전략, 보안 모두 적용.
```

```
NextJS-개발가이드.md를 참고해서 NextAuth.js 인증을 구현해줘.
middleware.ts로 보호 경로 설정, 서버/클라이언트 양쪽 세션 처리.
```

## 2. 크로스커팅 가이드 활용 프롬프트

### 디자인 시스템

```
디자인시스템-가이드.md를 참고해서 이 프로젝트의 디자인 토큰을 설정해줘.

```

```

각 컴포넌트는 변형(variant), 크기(size), 상태(disabled/loading)를 지원하고
접근성(WCAG 2.1 AA) 기준을 충족해야 해.
```

```

```
